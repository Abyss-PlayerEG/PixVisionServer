# MacOS命令

## 启动redis
```shell
brew services start redis
```

## 停止redis
```shell
brew services stop redis
```

## 启动mysql
```shell
brew services start mysql
 ```

## 停止mysql
```shell
brew services stop mysql
 ```

# Windows命令

## 启动redis
```shell
sudo net start redis
```

## 停止redis
```shell
sudo net stop redis
```

## 启动mysql
```shell
sudo net start mysql80
 ```

## 停止mysql
```shell
sudo net stop mysql80
 ```
