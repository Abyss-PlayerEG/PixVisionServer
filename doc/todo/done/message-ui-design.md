# 消息系统 UI 设计提示词

## 项目背景
为 PixVisionServer 消息系统设计一个聊天界面，采用左右分栏布局，类似主流即时通讯应用的风格。

## 整体布局

### 页面结构
```
+-----------------------------------------------+
|                   Header                      |
+-------------------+---------------------------+
|                   |                           |
|   消息列表区域     |      对话内容区域          |
|   (左侧)         |      (右侧)               |
|                   |                           |
+-------------------+---------------------------+
```

### 尺寸规范
- **页面最大宽度**：1200px，居中显示
- **左侧消息列表**：宽度 320px，固定宽度
- **右侧对话区域**：flex: 1，自适应剩余空间
- **页面高度**：100vh，填满整个视窗
- **间距**：区域间无间距，紧密排列

## 交互状态

### 会话列表交互
1. **悬停状态**：背景变为 #f5f7fa
2. **选中状态**：背景 #ecf5ff，左侧蓝色边框
3. **点击**：加载右侧聊天记录，标记已读

### 消息交互
1. **长按消息**：显示复制、撤回菜单
2. **点击头像**：显示用户资料卡片
3. **图片消息**：点击放大预览

### 输入交互
1. **Enter**：发送消息
2. **Shift + Enter**：换行
3. **输入为空**：发送按钮禁用
4. **发送中**：按钮显示加载状态

## 响应式设计

### 桌面端（>= 768px）
- 左右分栏显示
- 左侧固定 320px
- 右侧自适应

### 移动端（< 768px）
- 默认显示左侧列表
- 点击会话后，右侧全屏覆盖显示
- 左上角返回按钮返回列表

## 加载状态

### 消息加载中
- 骨架屏或加载动画
- 位置：消息列表顶部
- 触发：滚动到顶部时加载历史消息

### 发送消息中
- 消息气泡显示半透明
- 旁边显示加载图标
- 发送失败显示红色感叹号，点击重试

## 技术实现要点

1. **布局**：使用 Flexbox 实现左右分栏
2. **滚动**：消息列表使用 `overflow-y: auto` + 细滚动条
3. **动画**：消息出现使用 fadeIn 动画
4. **虚拟列表**：大量消息时使用虚拟滚动优化性能
5. **自动滚动**：新消息自动滚动到底部
6. **输入框自适应**：textarea 根据内容自动调整高度
